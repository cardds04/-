블로그 글쓰기 도우미

사진·단지·위치·키워드를 넣으면 메인 커버 이미지(동네·단지명 합성)와 챗GPT·제미나이용 프롬프트를 만들고, AI가 쓴 글을 한 번에 복사해 네이버 블로그에 붙여넣을 수 있게 정리합니다. (자동 게시·로그인 자동화는 하지 않습니다.)

2. 사진

첫 번째 줄의 사진이 메인 커버로 쓰입니다. 아래는 공간별 추가 사진입니다. 행 옆 「삭제」로 줄일 수 있습니다.

3. 메인 사진 미리보기 · 다운로드

왼쪽 하단에 위치(위) + 아파트명(아래)을 올립니다. 네이버 에디터에 이 이미지를 먼저 넣은 뒤 글을 이어 쓰면 됩니다.

4. AI(챗GPT·제미나이)용 프롬프트

  1. 위에서 고른 모든 사진을 채팅에 첨부합니다. (순서: 메인 → 아래 표의 순서대로)
  2. 「프롬프트 복사」 후 채팅에 붙여넣고 전송합니다.
  3. 나온 전체 답변을 5번 칸에 붙여넣습니다.

5. 최종 글 (AI 답변 붙여넣기 → 전체 복사)

네이버 스마트에디터는 HTML 붙여넣기가 제한될 수 있습니다. 그럴 때는 「전체 복사」로 메모장에 넣은 뒤 단락만 에디터에 옮기거나, Word에 붙였다가 다시 복사하는 방식이 안전합니다.
