프롬프트 · 설정

제목을 정하면 아래에 아이콘으로 쌓입니다. 누르면 이 칸에 불러옵니다.

체크 시 Grok 결과에 Topaz를 적용합니다. 체크 해제면 Grok mp4만 저장.

FFmpeg 경로는 topaz_preset.TOPAZ_FFMPEG_PATH로 고정입니다. 프리셋 구간은 TOPAZ_SEGMENT_DURATION_SEC(기본 6초, 앞부분만)로 조절합니다.

내보내기 명령에서 -filter_complex 뒤 따옴표 안 문자열만 붙여 넣으세요. 위 -vf 칸은 비웁니다.

Topaz 번들 FFmpeg에는 libx264가 없을 수 있음 → 맥은 h264_videotoolbox 권장.

② 영상만 Topaz로 출력 (Grok 안 씀)

이미 mp4/mov 등이 있을 때만 사용. 기본은 저장된 Topaz 내보내기 프리셋만 쓰므로 영상만 선택하면 됩니다. API 키·사진·프롬프트는 필요 없습니다.

영상 파일

Topaz 설정

FFmpeg 경로는 고정입니다. 구간 자르기는 topaz_preset.TOPAZ_SEGMENT_DURATION_SEC(기본 전체). 직접 필터를 쓰려면 아래 체크를 해제하세요.

예: tvai_fi=model=apo-8:... ,tvai_up=model=prob-4:... ,scale=w=1920:h=1080:... 한 줄. 위 -vf는 비움.

Topaz 번들 FFmpeg에는 libx264 없음. 맥 예: -c:v h264_videotoolbox -b:v 8M

tvai_fi 오류 시 서버가 내부에서 fps=24를 앞에 붙입니다. 환경 변수 GROK_TVAI_PREPEND_FPS로 조절(0이면 끔).
