클립 보정 — 왼쪽 썸네일 · 가운데 미리보기 · 오른쪽 톤 보정

배치: 썸네일(좌) → 미리보기(중) → 톤·색 보정(우). Shift+클릭 범위 선택 · Ctrl+클릭(macOS는 ⌘+클릭) 떨어진 클립 개별 토글 · Shift+Ctrl+클릭은 기존 선택에 범위 추가. 범위/다중 선택 시 슬라이더가 선택 클립 전체에 적용. 단축키(슬라이더·입력칸에 포커스가 있어도 동작): Ctrl+Z 실행 취소 · Ctrl+Shift+Z 다시 실행 · Alt+D 다음 클립 · Alt+S 이전 · Alt+W 스포이드 on/off. macOS는 ⌘+Z / ⌘+Shift+Z 도 동일합니다. 썸네일을 끌어 놓기(드래그)하면 순서가 바뀝니다. Delete / Backspace 또는 상단 선택 삭제로 제거(한글 입력 중·숫자 칸 포커스 시 Delete 제외). 변경 후 몽타주 GUI에서 「웹 저장값 불러오기」 하면 목록·순서가 반영됩니다. 저장 후 몽타주 GUI「웹 저장값 불러오기」→「만들기」 시 FFmpeg에 반영됩니다.

순서는 버튼 또는 드래그 · 서버 주소(http://127.0.0.1:…)로 열어야 합니다
